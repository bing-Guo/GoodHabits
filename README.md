# GoodHabits
一款可以協助使用者建立良好習慣的iOS App

## 介紹
1. 可以建立多個習慣
2. 每天完成後可以標記已完成
3. 藉由日曆查看完成進度
4. 可以移除不必要的習慣

## 畫面實作
純程式碼（FlexLayout+PinLayout）

## 使用套件
1. Realm
2. FlexLayout
3. PinLayout
4. JTAppleCalendar
